Pocket File is a simple program which allows you to transfer files directly from your phone to your pc (You can find it on Google Play Store at 0.80€). To send files you need an extra program to run on your pc
Win32 Download (You need java to run this)
Linux/Mac Download (You need java to run this)
Double click on the icon.
This is how the graphical user interface works
You need to open a "port" to let your phone connect to your pc.
A port is a number which identifies the different connections on your pc.
Leave at default if you do not need to change it.

WARNING: YOU CANNOT USE MULTIPLE CONNECTIONS ON THE SAME PORT !
Once clicked on "Listen" you will be able to start a connection with your phone.
You will also notice that the window title will have changed.
In front of the title there is your IP, a number identifying your pc in your network.


WARNING: ONLY IF YOU ARE CONNECTED TO YOUR NETWORK (i.e. your router) YOU CAN USE THIS IP.
If you are connected to an external network, you need to find your "global" IP.
To do this go on this site ---> whatismyipaddress.com
And grab the IP given. This ip will change if you restart your router !
Since you use an external network, YOU NEED TO OPEN THE PORT ON YOUR ROUTER.
To do this go on START --> ALL PROGRAMS --> ACCESSORIES --> COMMAND PROMPT.
Once opened, type ipconfig.
Find this line and take the GATEWAY NUMBER. It is normally 192.168.1.1
Type the gateway number on your browser. You will access to your router configuration
In "Port Forwarding" fill all the fields, the name of your application (Pocket File), the port you want to open and "TCP" as connection protocol.
Fill "IP" and "Port" fields on your phone with your local IP or with your global IP and with the port chosen
